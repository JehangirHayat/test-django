# AA Energies

AA Energies is a Django-based web application designed for energy monitoring and data visualization. It provides a comprehensive dashboard for tracking electrical parameters such as currents, voltages, power factors, and more, with support for data import/export, user authentication, and multi-language interfaces.

## Features

- **User Authentication**: Secure login system with role-based access (Admin, User, Viewer).
- **Dashboard**: Interactive dashboard with navigation to various data views.
- **Data Visualization**:
  - Energy Graphs (Enegia Grafikoa)
  - Power Graphs (Potentzia Grafikoa)
  - Dynamic Data Tables (Datos Dinamicos)
  - Constant Data Tables (Datos Constantes)
- **Data Management**: Import and export energy data in CSV, JSON, and XLSX formats.
- **Multi-Language Support**: Available in Spanish (es), English (en), and Basque (eu).
- **Themes**: Light and Dark mode support for better user experience.
- **Responsive Design**: Built with HTML, CSS, and JavaScript for web compatibility.

## Technologies Used

- **Backend**: Django 5.2.7
- **Database**: SQLite3
- **Frontend**: HTML, CSS, JavaScript
- **Libraries**: django-import-export for data handling
- **Internationalization**: Django's i18n framework
